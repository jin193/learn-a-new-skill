# 知径
一款针对技能，通过规划学习框架并够引导学习者通过自主探究学习完整技能的工具

## 过程
0：规划学习路径，构建技能知识框架：
· 学习者选择要学习的技能（如摄影、演讲、绘画等）
· 系统协助将技能分解为两层结构：
  · 第一层：技能的各个核心方面
  · 第二层：每个方面下的具体知识点
· 形成可视化的思维导图

1：生成知识点概述
· 针对即将学习的第二层知识点，系统自动生成一份简明概述，内容包括：
  · 该知识点的核心概念
  · 1-2个关键要点
  · 一个生活中的应用示例
· 概述以短段落、要点式的格式呈现，学习者可利用碎片化时间提前阅读，建立初步认知

2：正式学习
· 系统根据知识点概述，生成学习单，提出3-5个引导性问题，问题设计原则：
  · 开放性和探究性，鼓励思考
  · 需要学习者通过搜索、实践、观察等方式寻找答案
  · 与实际应用场景结合
· 学习者通过搜索引擎、实际操作、观察记录等方式寻找问题答案
· 上传文字等材料作为探究成果
· 学习者提交探究成果后，系统进行评审，内容包括：
  · 肯定正确的理解和有创意的思路
  · 指出理解偏差或遗漏的关键点
  · 补充相关知识或更好的示例
  · 提出可进一步思考的方向

3：练习与诊断
· 系统根据知识点生成5道选择题作为练习题，便于学习者作答
· 题目提交后即时批改，提供详细解析，说明正确选项的原因和各选项的分析
· 系统结合学习者练习中的表现，提炼出具体的的薄弱点
· 存储薄弱点，记录知识点、薄弱描述、掌握程度、发现时间


## 运行方式
1. 在 `backend/` 目录下打开终端

2. 安装依赖:
   - `pip install -r requirements.txt`

3. 设置环境变量:
   - `AI_API_KEY`
   - `AI_BASE_URL`  默认: https://api.deepseek.com
   - `AI_MODEL`     默认: deepseek-chat

   - `APP_HOST`     默认: 127.0.0.1
   - `APP_PORT`     默认: 2121

4. 运行服务:
   - `python app.py`
