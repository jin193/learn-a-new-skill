# 知径
一款针对技能，通过规划学习框架并够引导学习者通过自主探究学习完整技能的工具

## 过程
**1：规划学习框架**  
*   输入要学习的技能
*   形成可视化的思维导图

**2：预习概述**  
*   生成一份简明概述
*   提供纯内容页面

**3：正式学习**  
*   生成学习单，提出3-5个引导性问题
*   通过搜索引擎、实际操作等方式自主探究
*   输入回答 
*   评审回答

**4：练习诊断**
*   生成5道选择题
*   批改并详细解析
*   提炼薄弱点
*   存储薄弱点


## 运行方式
1. 在 `backend/` 目录下打开终端

2. 安装依赖:
   - `pip install -r requirements.txt`

3. 设置环境变量:
   - `AI_API_KEY`
   - `AI_BASE_URL`  默认: https://api.deepseek.com
   - `AI_MODEL`     默认: deepseek-chat

   - `APP_HOST`     默认: 127.0.0.1
   - `APP_PORT`     默认: 2121

4. 运行服务:
   - `python app.py`
