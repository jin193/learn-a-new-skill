任务：为一个技能设计一套具有过程性、循序渐进的学习知识框架

技能名称：{{skill}}

核心要求：
1. 两层结构：
   · 第一层（aspects）：技能学习的各个阶段或模块，必须按照从入门到进阶的推荐学习顺序排列（即第一个aspect是初学者最先需要掌握的，最后一个aspect是较高级的）
   · 第二层（topics）：每个阶段下的具体知识点，即一次学习能够掌握的具体技巧或概念。每个知识点应可描述为[具体动作/判断]。同一aspect内的topics也建议按学习顺序排列
2. 过程性设计原则：
   · 整个框架应构成一条自然的学习路径：从基础准备开始，逐步过渡到核心技能，最后到综合应用或进阶技巧
3. 知识点具体性标准：
   · 每个知识点必须指向可观察、可练习、可检验的具体能力
   · 内容需精炼，概括，aspects和其对应topics数量各不超过5个，topic字数不超过10字

输出格式：使用纯文本严格遵循以下JSON结构，不添加任何额外文字
{
  "skill": "...",
  "stages": [
    {
      "title": "...",
      "topics": ["...", "..."]
    }
  ]
}
